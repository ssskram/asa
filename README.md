## Installation

### Prerequisites
1. [.net sdk](https://www.microsoft.com/net/learn/get-started/) 

### Installation
1. Clone this repository in some place `~/nice`
2. Open `~/nice` in the terminal
3. Restore dependencies
```bash
dotnet restore
```
4. Build & run
```bash
dotnet build
dotnet run
```
